## 🔔 Notification Flow

Customer books:
1. Fills form on website
2. Clicks "Book Appointment"
3. Data sent to database
4. Admin sees in "Bookings" tab
5. Admin changes status to "Confirmed"
6. Customer gets notification (if email configured)
7. Customer sees appointment in profile (future feature)

Review submitted:
1. Customer leaves review
2. Goes to "Pending" in Reviews tab
3. Admin approves or rejects
4. If approved, shows on homepage
5. If rejected, stays hidden
