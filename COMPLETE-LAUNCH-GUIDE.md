## 🐛 QUICK FIXES

### Website won't start?
- Check Node.js: `node --version`
- If not found, install from nodejs.org
- Run script again

### Port 3000 error?
- Another app using the port
- Close other apps
- Or restart computer

### Can't login?
- Register first to create account
- Click "Register" button

### Services not showing?
- Refresh page: Ctrl+R
- Wait a moment

### Video won't play?
- Check internet connection
- URL must be .mp4
- Refresh page
