# 🚀 SETUP INSTRUCTIONS

Your Elite Parlor Website is ready! Follow these steps to get it running.

## ⚠️ FIRST: Install Node.js

This is required before anything else works.

### Windows:
1. Go to https://nodejs.org/
2. Click "LTS" (Long Term Support) - Green button
3. Download the installer
4. **RUN THE INSTALLER**
5. ✅ Check the box: "Add to PATH"
6. Complete the installation
7. **RESTART YOUR COMPUTER** (important!)
8. Open Command Prompt or PowerShell and verify:
   ```
   node --version
   npm --version
   ```

If you see version numbers, Node.js is installed! ✅

## 🐛 TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| "npm: command not found" | Node.js not installed. Install it first. |
| "Port 3000 already in use" | Close other apps using port 3000 |
| "Cannot find module" | Run `npm install` in that folder |
| Video won't play | Use direct .mp4 URL (test in browser first) |
| Website won't load | Make sure both servers are running |
| "Connection refused" | Check MongoDB - see below |

### MongoDB Setup

Your app needs a database. Choose ONE:

**Option A: Local MongoDB (Easiest)**
```bash
# Install MongoDB Community from: https://www.mongodb.com/try/download/community
# Start MongoDB service (Windows)
# Done!
```

**Option B: MongoDB Atlas (Cloud - Recommended)**
1. Go to https://www.mongodb.com/cloud/atlas
2. Create free account
3. Create free cluster
4. Get connection string
5. In `backend/.env` change:
   ```
   MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/parlor-website
   ```

## 🆘 STILL STUCK?

1. Check Node.js is installed: `node --version`
2. Check internet connection (for dependencies)
3. Check folder permissions (right-click > Properties > Security)
4. Try deleting `node_modules` and `package-lock.json`, then `npm install` again
5. Restart Command Prompt/PowerShell
