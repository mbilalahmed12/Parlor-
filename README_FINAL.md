## 🚀 HOW TO RUN (3 SIMPLE STEPS)

### Step 1: Check Node.js
Ensure Node.js is installed:
1. Open Command Prompt
2. Type: `node --version`
3. If you see a version number, skip to Step 2
4. If "not found", install from: https://nodejs.org/ (LTS)

### Step 2: Start Website
**Double-click: `run-website.bat`**

First time: Wait 10-15 minutes (installing dependencies)  
Later times: Waits 30 seconds (just starting servers)

### Step 3: Use Website
1. Browser opens to: http://localhost:3000
2. Click "Register" button
3. Create admin account (name, email, password)
4. You're logged in! ✅

**That's it!** ✨

## 🐛 TROUBLESHOOTING

**Website won't start?**
- Install Node.js from nodejs.org
- Restart Command Prompt
- Run script again

**"Port 3000 already in use"?**
- Another app is using port 3000
- Close other apps (browsers, development tools)
- Or change port in start command

**"Port 5000 already in use"?**
- Close any other backend servers
- Or change PORT in backend/.env

**Can't login?**
- Make sure you registered first
- Click "Register" to create admin account
- Check spelling of email

**Bookings not showing?**
- Refresh page (Ctrl+R)
- Check backend server is running (2nd command window)
- Check browser console (F12) for errors

**Video won't load?**
- URL must be .mp4 format
- URL must be direct link (not web page)
- Try videos from Pexels/Pixabay

**Services not showing?**
- Refresh page (Ctrl+R)
- Wait a moment and try again
- Check backend has no errors
