🎯 QUICK START (3 STEPS):

Step 1: Node.js Check
  - Open Command Prompt
  - Type: node --version
  - If you see version → Great! Skip to Step 2
  - If "not found" → Install from nodejs.org (LTS)

Step 2: Run Website
  - Double-click: run-website.bat
  - Wait 10-15 minutes (first time)
  - Later times: Just 30 seconds

Step 3: Use It!
  - Website opens at: http://localhost:3000
  - Click "Register"
  - Create admin account
  - Go to Admin Dashboard
  - Add your services
  - Customize settings
  - Done! ✨

🎬 CHANGING VIDEO BACKGROUND:

  1. Go to: Admin → Settings
  2. Find: Hero Video URL
  3. Go to: pexels.com/videos or pixabay.com/videos
  4. Search: "beauty", "salon", or "spa"
  5. Copy: .mp4 download link
  6. Paste: In Settings form
  7. Click: SAVE
  8. Done! Homepage updates instantly ✨

🐛 TROUBLESHOOTING:

Problem: "Node.js not found"
Solution: Install from nodejs.org (LTS version)

Problem: "Port 3000 already in use"
Solution: Close other apps or change port

Problem: "Can't login"
Solution: Register first to create account

Problem: "Bookings not showing"
Solution: Refresh page (Ctrl+R)

Problem: "Video won't load"
Solution: Use .mp4 URL only (Pexels/Pixabay)
