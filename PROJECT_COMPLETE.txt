## 🎯 QUICK START (3 STEPS)

### Step 1: Check Node.js
```
Open Command Prompt and type:
node --version

If you see a version → Great! Go to Step 2
If "not found" → Install Node.js from nodejs.org (LTS)
```

### Step 2: Run Website
```
Double-click: run-website.bat

First time: Wait 10-15 minutes (installing everything)
Later times: Just 30 seconds
```

### Step 3: Use It!
```
Website opens automatically at:
http://localhost:3000

Click "Register" to create admin account
Welcome to your parlor website! ✨
```

**Free video sources:**

- pixabay.com/videos
- unsplash.com

**Steps:**
1. Go to Settings tab
2. Paste video URL (.mp4 required)
3. Click Save
4. Homepage updates instantly! ✨
