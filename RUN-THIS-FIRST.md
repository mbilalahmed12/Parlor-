## 🐛 TROUBLESHOOTING

### "Node.js not found"
**Solution:** Install from nodejs.org, restart computer, try again

### "Port 3000 already in use"
**Solution:** Close other apps using port 3000, or restart computer

### "Backend connection error"
**Solution:** Wait a few seconds, refresh browser (Ctrl+R)

### "Services not showing"
**Solution:** Refresh page (Ctrl+R)

### "Can't login"
**Solution:** Register first to create account

### "Video won't play"
**Solution:** Wait a moment, then refresh. Check internet connection.
