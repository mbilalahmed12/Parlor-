## 🎬 VIDEO BACKGROUND

**Current Default:**
- Pexels beauty salon video
- Changeable from admin panel

**How to Change:**
1. Go to Admin Dashboard
2. Navigate to Settings tab
3. Find "Hero Video URL"
4. Paste new video URL (must be .mp4)
5. Click Save
6. Homepage refreshes with new video

**Free Video Sources:**
- Pexels.com/videos (high quality, free)
- Pixabay.com/videos (high quality, free)
- Unsplash.com (high quality, free)

## 🎯 READY TO USE?

### To Start Website:
1. Double-click `START.bat` or `run-website.bat`
2. Choose option 1
3. Wait 10-15 minutes for first-time setup
4. Website opens at http://localhost:3000

### First-Time:
1. Click "Register"
2. Create admin account
3. Go to Admin Dashboard
4. Add your services
5. Customize settings

### Share:
- Give customers the website URL
- They can book appointments
- You approve and manage from admin panel
