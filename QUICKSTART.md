# ⚡ Quick Start Guide

Get your parlor website running in 5 minutes!

## 🚀 Start the Servers

### Terminal 1 - Backend
```bash
cd backend
npm install
npm run dev
```
✅ Backend running on `http://localhost:5000`

### Terminal 2 - Frontend
```bash
cd frontend
npm install
npm run dev
```
✅ Frontend running on `http://localhost:3000`

## 📍 Links

| Link | Purpose |
|------|---------|
| `http://localhost:3000` | Website homepage |
| `http://localhost:3000/login` | Admin login |
| `http://localhost:3000/register` | Create admin account |
| `http://localhost:3000/admin` | Admin dashboard |
| `http://localhost:5000/api/health` | API health check |

## 👤 Create Your Account

1. Visit `http://localhost:3000/register`
2. Sign up as admin
3. You'll automatically go to dashboard

## 📋 5-Minute Setup Checklist

- [ ] Backend running (`npm run dev` in backend folder)
- [ ] Frontend running (`npm run dev` in frontend folder)
- [ ] Created admin account
- [ ] Added first service
- [ ] Updated hero video in Settings

## 🎥 Change Hero Video

1. Go to Admin Dashboard
2. Click "Settings"
3. Find "Hero Video URL"
4. Paste any direct .mp4 link (e.g., from Pexels)
5. Click "Save All Changes"

## ➕ Add Services

1. Admin > Services
2. Click "Add Service"
3. Fill details
4. Click "Create"

## 📅 Manage Bookings

1. Admin > Bookings
2. View all client bookings
3. Confirm or delete bookings
4. Track booking status

## ⭐ Approve Reviews

1. Admin > Reviews
2. See pending reviews
3. Click "Approve & Publish" to show on website
4. Or delete inappropriate reviews

## 🎨 Customize Website

**Settings** tab controls:
- Website name & description
- Homepage video & text
- Contact phone/email/address
- Social media links
- Business hours

## 🆘 Common Issues

| Issue | Solution |
|-------|----------|
| Backend won't start | Check MongoDB is running |
| Can't see services | Make sure backend is running |
| Video won't play | Use direct .mp4 URL |
| Can't login | Check email/password, try register |
